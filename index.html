Headers

It is usually a good idea to separate text content by topic. A good way to do that is with headers. There are six sizes of header to work with, starting with Header1 - the largest - and ending with Header6, the smallest. The html element syntax for headers is:

Lists

Another useful tool is a List. In basic HTML, there are two kinds of lists to consider. The first is the Un-Ordered List. The items place in an un-ordered list are separated by bullet points. The syntaxt for un-ordered lists uses opening and closing tags to establish the list, then opening and closing list tags for the items in the list.

The opening and closing list tags look like this:

<ul></ul>

The opening and closing list item tags look like this:

<li></li>

A simple un-ordered list containing a single item, "List Item", might look like this in HTML:

<ul>

<li>List Item</li>

</ul>

In a web browser, the un-ordered list looks like this:

  • Here is my first list item
  • Here is another.
  • You get the point

Another kind of list is the Ordered List. In an ordered list, the listed items are numbered. That's the only difference! The code looks like this:

<ol>

<li>List Item</li>

</ol>

In a web browser, the ordered list looks like this:

  1. Here is my first ordered item
  2. Here's another.
  3. Last but not least

Paragraphs

This one is easy. The paragraph elment has opening and closing tags, and can contain other text-modifying elements within it. It can also be styled directly or in a CSS file using the same methods as other HTML elements.

The code for a paragraph looks like this:

<p>The paragraph text goes here</p>

Emphasis can be added to text by using the strong and emphasis (italics) tags. The code looks like this:

<strong>This text would be BOLD</strong>

<em>This text would be in italics</em>

The last tag to discuss in this section is the "break" tag. Its job
is to act
like a return
button.

There is no closing tag for a break. It looks like this in code:

<br>

Ordinary text can be very informative, even entertaining, but most consumers of web content expect at least some visually interesting content in the form of images and video from the pages they frequent. Videos

It's easy to add .mp4 format video files to your web page. Just like the image element, the video element uses the "src" to designate the "source" file to display. Unlike the image element, the video uses designated width and height dimensions. If desired, adding "controls" within the opening tag will display video controls to the user. Here is a video I once used in a school project:

Here is what the code looks like:

<video src="./Image_Folder/FileName.mp4" width="320" height="240" controls></video>

Notice that the video element includes a closing tag (unlike the image element).
